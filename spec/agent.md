# Agent Entry

## 你是新進來的 agent 時，先這樣讀
1. 讀 [PROJECT_MAP.md](./PROJECT_MAP.md)
2. 讀 [architecture/system-overview.md](./architecture/system-overview.md)
3. 若任務碰 UI，讀 [modules/world-frontend.md](./modules/world-frontend.md)
4. 若任務碰資料 / API，讀 [modules/runtime-backend.md](./modules/runtime-backend.md)
5. 若任務碰視覺重構，讀 [conventions/visual-and-doc-standards.md](./conventions/visual-and-doc-standards.md)

## 不要直接做的事
- 不要先整庫暴力掃描
- 不要只因為 Star Office 可視化有效，就把它一比一複製進來
- 不要把新規則只寫在聊天，忘了同步 spec

## 這個專案的首要判準
- 可讀性勝過裝飾密度
- 主代理勝過次要面板
- 穩定資料契約勝過短期拼接技巧
