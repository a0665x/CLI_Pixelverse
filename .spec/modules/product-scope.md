# Product Scope

## MVP
- 1 個主 Hermes agent 像素角色
- 顯示狀態：idle / thinking / working / speaking / sleeping / error
- 顯示目前任務摘要
- 顯示已知 subagents / branches 的像素角色
- 點擊角色可看詳細資訊
- 工具圖示列：最近或正在使用的工具
- 可透過設定檔自訂角色名稱、顏色、世界主題、圖示映射

## Phase 2
- 真正的工具逐步事件流
- 角色路徑動畫與不同工作站
- 根據工具類型切換場景/動作
- 多個 Hermes profile / 多個主 agent
- 世界編輯器與 sprite/theme pack

## Out of Scope for First Pass
- 完整複製 miniverse 全部 world-editing 系統
- 遠端多使用者權限系統
- 把所有 Hermes CLI 互動都做成 100% 可回放
