# Office Asset Direction

Pixelverse uses built-in 32x32 top-down pixel SVG furniture for the default UI so the repo stays self-contained.

Optional visual reference:

- Pixel Life: Office Essentials
- Author: Chris Perich
- Source: https://christianperich.itch.io/pixel-life-office-essentials
- License shown on itch.io: Creative Commons Attribution 4.0 International

Do not commit paid asset zip contents or redistribute standalone downloaded files. If a local deployment owns the pack, it can replace the built-in fallback furniture sprites while keeping attribution in downstream docs.

Bundled free furniture source:

- Modern Interior Tileset by AppleDog
- Author: Apple Dog
- Source: https://apple-dog.itch.io/modern-tileset-by-appledog
- Author terms shown on itch.io comments: free for commercial/non-commercial projects with Apple Dog credited

Pixelverse currently uses selected 32x32 furniture crops from this pack for beds, sofas, chairs, desks, tables, file cabinets, lamps, and terminal-like office equipment. The pack does not provide a complete agent character animation set for this UI, so agent sprites remain separate.

Reference-only source inspected on May 25, 2026:

- Modern Interiors - RPG Tileset [16X16]
- Author: LimeZu
- Source: https://limezu.itch.io/moderninteriors
- Local free pack inspected at `/home/a0665x/Downloads/Modern tiles_Free/`
- Free pack license states non-commercial use only. Do not bundle it into this repo unless the project is explicitly non-commercial, or a paid license is available.

The current redesign borrows the room-planning direction from this pack but does not copy LimeZu free assets into `public/assets`.
